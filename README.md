# 🛒 E-Commerce Sales & Customer Analytics Portfolio

### 📌 Project Overview
This project provides a comprehensive analysis of a retail e-commerce dataset. The goal was to transform raw transactional data into actionable business intelligence, focusing on revenue trends, product performance, and customer demographics.

---

### 🛠️ Tech Stack
* **SQL:** Relational data modeling and advanced querying for sales metrics.
* **Excel:** Initial data cleaning, deduplication, and ETL (Extract, Transform, Load) processes.
* **Power BI:** Interactive reporting for C-suite stakeholders.
* **Looker Studio:** Cloud-based visualization for real-time sales tracking.

---

### 🏗️ Data Architecture (SQL)
To ensure high performance and scalability, the data was organized into a **Star Schema**:
* **Fact Table:** `Sales_Transactions` (The central hub containing quantities and prices).
* **Dimension Tables:** * `Dim_Products`: Category-level details.
    * `Dim_Customers`: Geographical and demographic data.
    * `Dim_Calendar`: Time-series data for trend analysis.

---

### 📈 Business Insights Delivered
1. **Sales Seasonality:** Identified peak purchasing windows, allowing for optimized inventory management.
2. **Customer Segmentation:** Analyzed high-value customer segments to help target marketing spend more effectively.
3. **Category Performance:** Discovered underperforming product lines with high churn rates, prompting a review of pricing strategies.
